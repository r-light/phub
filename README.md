# phub

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
![GitHub release (latest by date)](https://img.shields.io/github/v/release/r-light/phub)

一个简单的视频客户端，使用 flutter 开发。

## 实现功能

- [x] 搜索
- [x] 收藏
- [x] 历史记录
- [x] 自动获取临时域名
- [ ] 长按加速
- [ ] 播放器翻译
